Introduction and Big O Notation

Introduction

  • Algorithm: a sequence of computational steps that transform the input in the output.
  • Instance of a problem: consists of the input, satisfying the constraints imposed by the problem, needed to compute a solution of the problem.
  • Data structure: is a way to store and organize data in order to facilitate access and modifications.

An example: Insertion Sort

  • Input: A sequence of n numbers <a1, a2, ..., an>
  • Output: a permutation (reordering) of the input sequence such that the sequence is now ordered Note: the number that we wish to sort are also referenced as keys.

This problem is approached the same as one would order the cards in his hands. It would place a card at the time, placing the card in the correct ordered place given the already placed cards. 

def insertion_sort(a: array):
    for j = 1 to a.size():
         key = a[j]
         i = j - 1
         while i > 0 and a[i] > key:
             a[i + 1] = a[i]
             i = i - 1
         a[i + 1] = key

  • In line 5-6 we move any card that proceed the new taken card 1 position next if it is bigger than the new taken card.
